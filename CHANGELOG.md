# Changelog

## [1.0.1] - 2025-12-30

### Added
- デフォルトロック設定機能（設定画面からON/OFF切り替え可能）
- 投稿一覧での一括ロック/解除機能（バルクアクション）
- 設定画面から全投稿を一括ロック/解除する機能

### Changed
- 設定メニューをトップレベルメニューに変更（位置81）

## [1.0.0] - 2025-10-26

### Added
- 更新日ロック機能の実装
- 手動更新日変更機能の実装
- 投稿タイプ選択機能
- メタボックスUI
- AJAX処理による日時変更
- 「公開日と同じにする」ボタン
- デフォルトロック機能
- プラグイン一覧に設定リンク追加

### Fixed
- ロック機能が正しく動作しない問題を修正
- フィルターの優先度を調整

### Improved
- モジュラー設計への分割
- セキュリティ強化（nonce検証）
- コードの最適化
